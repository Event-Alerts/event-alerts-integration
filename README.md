# Event Alerts Integration

A plugin to integrate your Minecraft server with the Event Alerts ecosystem

**🐛 Bugs / 💡 Suggestions:** Please [open an issue](https://github.com/Event-Alerts/event-alerts-integration/issues/new/choose) to report a bug or suggest an idea

**🆘 Support:** Please [join the Discord](https://eventalerts.gg/discord) to get support

## Download

**✅ Stable:** You can download the latest **stable** version at [Modrinth](https://modrinth.com/plugin/event-alerts-integration) or [GitHub](https://github.com/Event-Alerts/event-alerts-integration/releases)

**🚧 Snapshot:** You can download the latest **snapshot** version at [github.com/Event-Alerts/event-alerts-integration/actions/workflows/build.yml](https://github.com/Event-Alerts/event-alerts-integration/actions/workflows/build.yml)
